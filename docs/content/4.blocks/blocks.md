# Blocks

Collection of reusable UI components and blocks built with Nuxt Motion Block, Nuxt UI, and motion-v. These blocks are ready to use in your projects with minimal customization required.

## Hero Sections

Create stunning hero sections with animated backgrounds and engaging content.

- [Gradient Hero](/blocks/hero-sections/gradient-hero) - Modern hero section with animated gradient background

## Navigation

Responsive navigation components with dropdown support and scroll effects.

- [Modern Navbar](/blocks/navigation/modern-navbar) - Responsive navbar with dropdown menus

## Features

Showcase your product capabilities with beautifully designed feature sections.

- [Feature Grid](/blocks/features/feature-grid) - Responsive grid layout for feature highlights

## Testimonials

Display customer reviews and feedback with elegant testimonial sections.

- [Testimonial Carousel](/blocks/testimonials/testimonial-carousel) - Mobile-friendly testimonial carousel

## Pricing

Present your subscription plans and packages with clear pricing sections.

- [Pricing Cards](/blocks/pricing/pricing-cards) - Tiered pricing with feature comparisons

## Call to Action

Drive user engagement and conversions with compelling CTA sections.

- [Centered CTA](/blocks/cta/centered-cta) - Centered call-to-action with background effects

## Footers

Comprehensive footer components with multiple columns and social links.

- [Multi-column Footer](/blocks/footers/multi-column-footer) - Full-featured footer with navigation

## Featured Components

Showcase of individual components from the nuxt-motion-block library.

- [Animated Spotlight](/blocks/featured-components/animated-spotlight) - Interactive spotlight effect that follows the cursor
- [Animated Gradient](/blocks/featured-components/animated-gradient) - Dynamic gradient background with smooth transitions
- [Ripple Effect](/blocks/featured-components/ripple-effect) - Calming ripple effect for backgrounds


## Usage

All blocks are built using the nuxt-motion-block components and can be easily customized to match your brand. Simply copy the code examples and adjust the content and styling to fit your needs.
